# 3. What is Linear Algebra?

Linear algebra is:

> A mathematical framework for solving systems of linear equations.

---

## 3.1 Example: Sheriff and Bank Robber

Imagine:

* Sheriff car = one line
* Bank robber car = another line

Where the lines intersect:

* Gives the solution
* Represents matching time and distance

---

## 3.2 Possible Outcomes in Linear Algebra

There are only 3 possibilities:

| Situation        | Meaning            |
| ---------------- | ------------------ |
| One intersection | One solution       |
| Parallel lines   | No solution        |
| Same line        | Infinite solutions |

Important:

* Linear equations cannot intersect multiple times
* Therefore, systems cannot have exactly 2 or 3 solutions

# 8. Norms

Norms measure vector size (magnitude).

Most important:

* L2 Norm (Euclidean norm)

---

## Formula

|x|_2 = \sqrt{x_1^2 + x_2^2 + \cdots + x_n^2}

---

## NumPy Example

-------------------code----------------------
import numpy as np

x = np.array([3, 4])

norm = np.linalg.norm(x)

print(norm)
-------------------code----------------------

Output:

-------------------code----------------------
5.0
-------------------code----------------------

---

# 10. Orthogonal and Orthonormal Vectors

## Orthogonal Vectors

Two vectors are orthogonal if:

* Their dot product equals zero

---

## Formula

x \cdot y = 0

---

## Example

-------------------code----------------------
import numpy as np

x = np.array([1, 0])
y = np.array([0, 1])

print(np.dot(x, y))
-------------------code----------------------

Output:

-------------------code----------------------
0
-------------------code----------------------

---

## Orthonormal Vectors

Vectors are orthonormal if:

1. They are orthogonal
2. Each has unit norm

---

# 12. Symmetric Matrices

A matrix is symmetric if:

* It is square
* Transpose equals itself

---

## Formula

---

## Example

-------------------code----------------------

    [1, 2],
    [2, 3]
])

-------------------code----------------------

-------------------code----------------------
True

# 13. Matrix Multiplication

One of the most important operations in ML.

Condition:

* Columns of first matrix
* Must equal rows of second matrix

---

## Formula

---

    [1, 2],
    [3, 4]

    [5],
    [6]

# 16. Matrix Inversion Limitations

Matrix inversion only works if:

1. Matrix is square
2. Matrix is non-singular
3. Columns are linearly independent

---

## Singular Matrix Example

-------------------code----------------------
import numpy as np

X = np.array([
    [1, 2],
    [2, 4]
])

np.linalg.inv(X)
-------------------code----------------------

This produces:

-------------------code----------------------
LinAlgError: Singular matrix
-------------------code----------------------

Because:

* Second row is a multiple of first row

# 18. Orthogonal Matrices

Orthogonal matrices contain:

* Orthonormal rows
* Orthonormal columns

## Important Property

## Another Important Property

For orthogonal matrices:

# 20. Important Notes and Best Practices

## Matrix Multiplication

Always verify dimensions:

-------------------code----------------------
(m × n) · (n × p)
-------------------code----------------------

Result:

-------------------code----------------------
(m × p)
-------------------code----------------------

---

## Matrix Inversion

Avoid direct inversion in real ML systems when possible.

Instead use:

* Pseudoinverse
* Numerical solvers
* Factorization methods

They are:

* Faster
* More stable
* More memory efficient

---

## Orthogonal Matrices

Orthogonal matrices are valuable because:

* Preserve lengths
* Preserve angles
* Inversion is computationally cheap

Widely used in:

* PCA
* Computer graphics
* Deep learning
* Signal processing
