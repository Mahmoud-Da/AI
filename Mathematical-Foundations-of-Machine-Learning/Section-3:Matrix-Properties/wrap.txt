# 4. Topics Covered in Segment 3

## 4.1 Frobenius Norm

* Measures the **size (magnitude)** of a matrix

### Formula

|X|*F = \sqrt{\sum*{i,j} x_{ij}^2}

### Key Idea

* Extension of L2 norm from vectors → matrices

---

## 4.2 Matrix Multiplication

### Types:

1. Matrix × Vector
2. Matrix × Matrix

### Key Idea

* Core operation in neural networks
* Used in every layer of deep learning models

---

## 4.3 Special Types of Matrices

### 4.3.1 Symmetric Matrix

* Equal to its transpose

[
X = X^T
]

---

### 4.3.2 Identity Matrix

* Acts like the number 1 in matrix multiplication

[
XI = X
]

---

## 4.4 Matrix Inversion

### Definition

[
X^{-1}
]

### Key Idea

* Solves linear systems efficiently
* Extremely important in machine learning
* Often computationally expensive

---

## 4.5 Additional Special Matrices

### 4.5.1 Diagonal Matrix

* Only diagonal elements are non-zero

### 4.5.2 Orthogonal Matrix

* Columns are orthonormal
* Transpose equals inverse

[
X^T = X^{-1}
]

# 6. Practical Note (No Code in This Lecture)

This lecture is conceptual (roadmap), but here is a simple example showing how matrices are used in code:

-------------------code----------------------
import numpy as np

# Matrix
X = np.array([[1, 2],
              [3, 4]])

# Vector
v = np.array([5, 6])

# Matrix-vector multiplication
result = X.dot(v)

print(result)
-------------------code----------------------
