# 4. Singular Value Decomposition (SVD)

## Why SVD Is Needed

Eigen decomposition only works for:

* Square matrices

[
n \times n
]

However, real-world datasets are often:

[
m \times n
]

where:

[
m \neq n
]

These are rectangular matrices.

---

## What SVD Does

SVD generalizes matrix decomposition to any matrix shape.

[
A = U\Sigma V^T
]

where:

* (U) = left singular vectors
* (\Sigma) = singular values
* (V^T) = right singular vectors

---

## Why It Matters

Most real-world machine learning datasets are rectangular.

Examples:

* Images
* User-item recommendation matrices
* Tabular datasets
* Text embeddings

SVD is one of the most important tools in machine learning.

---

## Practical Application

The lecture previews a demonstration where SVD is used to:

* Compress an image
* Reduce image size by 96%

while preserving most visual information.

---

## Basic SVD Example

-------------------code----------------------
import numpy as np

A = np.array([
    [1, 2],
    [3, 4],
    [5, 6]
])

U, S, VT = np.linalg.svd(A)

print(U)
print(S)
print(VT)
-------------------code----------------------

# 5. Moore-Penrose Pseudoinverse

## Motivation

Ordinary matrix inversion requires:

1. Square matrix
2. Non-singular matrix

Many machine learning problems violate these conditions.

---

### Example

A dataset might have:

[
1000 \times 10
]

matrix dimensions.

This matrix cannot be inverted directly because it is not square.

---

## Solution

Use the Moore-Penrose Pseudoinverse.

Notation:

[
A^+
]

---

## Why It Matters

The pseudoinverse allows us to:

* Solve systems of equations
* Estimate unknown values
* Perform regression

even when matrices are not invertible.

---

## Practical Application

The lecture previews fitting a line to data points using the pseudoinverse.

This is the foundation of:

* Linear Regression
* Least Squares Optimization

---

## Example Code

-------------------code----------------------
import numpy as np

A = np.array([
    [1, 1],
    [2, 1],
    [3, 1]
])

A_pinv = np.linalg.pinv(A)

print(A_pinv)
-------------------code----------------------

# 6. Trace Operator

## What Is the Trace?

The trace of a square matrix is the sum of the diagonal entries.

Notation:

[
tr(A)
]

---

### Example

[
A=
\begin{bmatrix}
1 & 2\
3 & 4
\end{bmatrix}
]

Trace:

[
tr(A)=1+4=5
]

---

## Python Example

-------------------code----------------------
import numpy as np

A = np.array([
    [1, 2],
    [3, 4]
])

print(np.trace(A))
-------------------code----------------------

# 7. Principal Component Analysis (PCA)

## Goal

PCA identifies important patterns in data.

---

### What Makes PCA Special?

PCA works even when data is:

* Unlabeled
* High-dimensional
* Difficult to visualize

---

### PCA Finds

1. Directions of maximum variation
2. Most informative features
3. Lower-dimensional representations

---

## Applications

* Data compression
* Visualization
* Noise reduction
* Feature extraction

    [3, 6],

*** 43. Singular Value Decomposition ***
# Singular Value Decomposition (SVD) — Structured Notes

## 1. Introduction to Singular Value Decomposition (SVD)

### What is SVD?

**Singular Value Decomposition (SVD)** is a matrix decomposition technique widely used in:

* Machine Learning
* Data Science
* Computer Vision
* Recommendation Systems
* Data Compression

Unlike **Eigen Decomposition**, which only works on **square matrices**, SVD works on **any real-valued matrix**, including rectangular matrices.

### Key Advantage

| Eigen Decomposition           | Singular Value Decomposition          |
| ----------------------------- | ------------------------------------- |
| Only square matrices          | Any real-valued matrix                |
| Uses eigenvectors/eigenvalues | Uses singular vectors/singular values |
| More restrictive              | More general                          |

---

# 2. SVD Formula

For a matrix:

[
A
]

Its Singular Value Decomposition is:

[
A = UDV^T
]

Where:

### U Matrix

[
U \in \mathbb{R}^{m \times m}
]

* Orthogonal matrix
* Columns are **Left Singular Vectors**
* Number of columns equals number of rows in A

---

### D Matrix

[
D \in \mathbb{R}^{m \times n}
]

* Diagonal matrix
* Contains singular values on the main diagonal
* All other entries are zero

---

### Vᵀ Matrix

[
V^T \in \mathbb{R}^{n \times n}
]

* Orthogonal matrix
* Rows are **Right Singular Vectors**
* Number of columns equals number of columns in A

---

# 3. Example Matrix

The lecture uses a non-square matrix:

[
A=
\begin{bmatrix}
1 & 2\
3 & 4\
5 & 6
\end{bmatrix}
]

Dimensions:

* Rows = 3
* Columns = 2

Therefore:

### U

[
3 \times 3
]

### D

[
3 \times 2
]

### V^T

[
2 \times 2
]

---

# 4. Computing SVD in NumPy

## Step 1: Create Matrix A

-------------------code----------------------
import numpy as np

A = np.array([
    [1, 2],
    [3, 4],
    [5, 6]
])
-------------------code----------------------

---

## Step 2: Compute SVD

-------------------code----------------------
U, d, VT = np.linalg.svd(A)
-------------------code----------------------

### Returned Objects

#### U

-------------------code----------------------
print(U)
-------------------code----------------------

Contains:

* Left Singular Vector 1
* Left Singular Vector 2
* Left Singular Vector 3

(each column is a singular vector)

---

#### Singular Values

-------------------code----------------------
print(d)
-------------------code----------------------

Example output:

-------------------code----------------------
[9.5255 0.5143]
-------------------code----------------------

These are the singular values.

---

#### Vᵀ

-------------------code----------------------
print(VT)
-------------------code----------------------

Contains right singular vectors as rows.

---

# 5. Building the Diagonal Matrix D

The SVD function returns singular values as a vector.

We must place them along the diagonal.

## Step 1: Create Diagonal Matrix

-------------------code----------------------
D = np.diag(d)
-------------------code----------------------

Result:

-------------------code----------------------
[
 [9.52, 0],
 [0, 0.51]
]
-------------------code----------------------

But dimensions are:

[
2 \times 2
]

We need:

[
3 \times 2
]

---

## Step 2: Expand D

-------------------code----------------------
D = np.vstack([
    D,
    [0, 0]
])
-------------------code----------------------

Result:

-------------------code----------------------
[
 [9.52, 0],
 [0, 0.51],
 [0, 0]
]
-------------------code----------------------

Dimensions:

[
3 \times 2
]

Now matrix multiplication is possible.

---

# 6. Reconstructing the Original Matrix

After decomposition:

[
A = UDV^T
]

We can verify this.

## Step 1: Multiply D and Vᵀ

-------------------code----------------------
DV = np.dot(D, VT)
-------------------code----------------------

---

## Step 2: Multiply U and (DV)

-------------------code----------------------
A_reconstructed = np.dot(U, DV)
-------------------code----------------------

---

## Step 3: Display Result

-------------------code----------------------
print(A_reconstructed)
-------------------code----------------------

Output:

-------------------code----------------------
[
 [1. 2.]
 [3. 4.]
 [5. 6.]
]
-------------------code----------------------

The original matrix is recovered.

---

# 7. Understanding the Components

## Left Singular Vectors (U)

Columns of U.

Example:

-------------------code----------------------text
u1
u2
u3
-------------------code----------------------

Represent important directions in row space.

---

## Right Singular Vectors (V)

Columns of V.

Represent important directions in column space.

Since NumPy returns:

-------------------code----------------------
VT
-------------------code----------------------

the singular vectors appear as rows.

---

## Singular Values (D)

Diagonal entries:

-------------------code----------------------text
σ1
σ2
...
-------------------code----------------------

Measure the importance of each singular vector.

Larger singular values:

* Carry more information
* Capture more structure in the data

Smaller singular values:

* Carry less information
* Often removed during compression

---

# 8. Relationship Between SVD and Eigen Decomposition

SVD and Eigen Decomposition are closely related.

---

## Left Singular Vectors

The left singular vectors of A are the eigenvectors of:

[
AA^T
]

[
\text{Left Singular Vectors}
============================

\text{Eigenvectors}(AA^T)
]

---

## Right Singular Vectors

The right singular vectors of A are the eigenvectors of:

[
A^TA
]

[
\text{Right Singular Vectors}
=============================

\text{Eigenvectors}(A^TA)
]

---

## Singular Values

Non-zero singular values are:

[
\sigma_i
========

\sqrt{\lambda_i}
]

where:

[
\lambda_i
]

are eigenvalues of:

[
A^TA
]

or

[
AA^T
]

---

# 10. Important Notes

### Orthogonal Matrices

Both U and V are orthogonal:

[
U^TU = I
]

[
V^TV = I
]

---

### Singular Values Are Non-Negative

Unlike eigenvalues:

* Eigenvalues can be negative.
* Singular values are always ≥ 0.

---

### SVD Always Exists

For any real-valued matrix:

[
A
]

an SVD decomposition exists.

This makes SVD one of the most powerful decomposition techniques.

---

# 11. Best Practices

### Use NumPy's Built-in SVD

-------------------code----------------------
np.linalg.svd()
-------------------code----------------------

instead of implementing SVD manually.

---

### Verify Reconstruction

Always check:

-------------------code----------------------
U @ D @ VT
-------------------code----------------------

matches the original matrix.

---

### Understand Dimensions

For:

[
A_{m \times n}
]

Dimensions are:

| Matrix | Size  |
| ------ | ----- |
| U      | m × m |
| D      | m × n |
| Vᵀ     | n × n |

Dimension mismatches are a common source of errors.

---

### Keep Largest Singular Values

For compression and dimensionality reduction:

* Keep largest singular values.
* Discard small singular values.

This preserves most information while reducing storage.

---

# 12. Full Final Code

-------------------code----------------------
import numpy as np

# Create matrix
A = np.array([
    [1, 2],
    [3, 4],
    [5, 6]
])

# Compute SVD
U, d, VT = np.linalg.svd(A)

print("U:")
print(U)

print("\nSingular Values:")
print(d)

print("\nVT:")
print(VT)

# Create diagonal matrix
D = np.diag(d)

# Expand to match dimensions (3x2)
D = np.vstack([
    D,
    [0, 0]
])

print("\nD:")
print(D)

# Reconstruct matrix
A_reconstructed = U @ D @ VT

print("\nReconstructed A:")
print(A_reconstructed)

# Verify equality
print("\nOriginal A:")
print(A)
-------------------code----------------------

# 13. Summary

### SVD Definition

[
A = UDV^T
]

where:

* **U** = Left Singular Vectors
* **D** = Singular Values
* **Vᵀ** = Right Singular Vectors

### Key Concepts

* Works on any real-valued matrix.
* Generalizes Eigen Decomposition.
* U and V are orthogonal matrices.
* Singular values measure importance of directions in the data.
* Original matrix can be perfectly reconstructed from U, D, and Vᵀ.

### Relationship to Eigen Decomposition

* Left singular vectors = eigenvectors of (AA^T)
* Right singular vectors = eigenvectors of (A^TA)
* Singular values = square roots of eigenvalues

### Machine Learning Importance

SVD is foundational for:

* PCA
* Dimensionality Reduction
* Recommendation Systems
* Image Compression
* Noise Filtering
* Latent Semantic Analysis
