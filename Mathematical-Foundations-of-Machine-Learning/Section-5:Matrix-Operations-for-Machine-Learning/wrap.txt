### Key Ideas Learned

#### Eigenvectors

Special vectors whose direction remains unchanged after a matrix transformation.

[
Av = \lambda v
]

---

#### Eigenvalues

Scalars that indicate:

* Stretching
* Compression
* Reflection

along an eigenvector direction.

---

#### Determinants

Special scalar values that:

* Measure volume scaling
* Indicate invertibility
* Relate directly to eigenvalues

[
\det(A)=\prod \lambda_i
]

---

#### Eigen Decomposition

Decomposing a matrix into:

[
A = V\Lambda V^{-1}
]

or for symmetric matrices:

[
A = Q\Lambda Q^T
]

This allows us to understand a matrix through its fundamental characteristics.

# 4. Singular Value Decomposition (SVD)

## Why SVD Is Needed

Eigen decomposition only works for:

* Square matrices

[
n \times n
]

However, real-world datasets are often:

[
m \times n
]

where:

[
m \neq n
]

These are rectangular matrices.

---

## What SVD Does

SVD generalizes matrix decomposition to any matrix shape.

[
A = U\Sigma V^T
]

where:

* (U) = left singular vectors
* (\Sigma) = singular values
* (V^T) = right singular vectors

---

## Why It Matters

Most real-world machine learning datasets are rectangular.

Examples:

* Images
* User-item recommendation matrices
* Tabular datasets
* Text embeddings

SVD is one of the most important tools in machine learning.

---

## Practical Application

The lecture previews a demonstration where SVD is used to:

* Compress an image
* Reduce image size by 96%

while preserving most visual information.

---

## Basic SVD Example

-------------------code----------------------
import numpy as np

A = np.array([
    [1, 2],
    [3, 4],
    [5, 6]
])

U, S, VT = np.linalg.svd(A)

print(U)
print(S)
print(VT)
-------------------code----------------------

# 5. Moore-Penrose Pseudoinverse

## Motivation

Ordinary matrix inversion requires:

1. Square matrix
2. Non-singular matrix

Many machine learning problems violate these conditions.

---

### Example

A dataset might have:

[
1000 \times 10
]

matrix dimensions.

This matrix cannot be inverted directly because it is not square.

---

## Solution

Use the Moore-Penrose Pseudoinverse.

Notation:

[
A^+
]

---

## Why It Matters

The pseudoinverse allows us to:

* Solve systems of equations
* Estimate unknown values
* Perform regression

even when matrices are not invertible.

---

## Practical Application

The lecture previews fitting a line to data points using the pseudoinverse.

This is the foundation of:

* Linear Regression
* Least Squares Optimization

---

## Example Code

-------------------code----------------------
import numpy as np

A = np.array([
    [1, 1],
    [2, 1],
    [3, 1]
])

A_pinv = np.linalg.pinv(A)

print(A_pinv)
-------------------code----------------------

---

# 6. Trace Operator

## What Is the Trace?

The trace of a square matrix is the sum of the diagonal entries.

Notation:

[
tr(A)
]

---

### Example

[
A=
\begin{bmatrix}
1 & 2\
3 & 4
\end{bmatrix}
]

Trace:

[
tr(A)=1+4=5
]

---

## Python Example

-------------------code----------------------
import numpy as np

A = np.array([
    [1, 2],
    [3, 4]
])

print(np.trace(A))
-------------------code----------------------

---

## Why Trace Matters

The trace operator becomes important for:

* Matrix algebra
* Optimization
* Principal Component Analysis (PCA)

It is one of the final pieces of linear algebra required before PCA.

# 7. Principal Component Analysis (PCA)

## Goal

PCA identifies important patterns in data.

---

### What Makes PCA Special?

PCA works even when data is:

* Unlabeled
* High-dimensional
* Difficult to visualize

---

### PCA Finds

1. Directions of maximum variation
2. Most informative features
3. Lower-dimensional representations

---

## Applications

* Data compression
* Visualization
* Noise reduction
* Feature extraction
* Machine learning preprocessing

---

## Simple Example

-------------------code----------------------
from sklearn.decomposition import PCA
import numpy as np

X = np.array([
    [1, 2],
    [2, 4],
    [3, 6],
    [4, 8]
])

pca = PCA(n_components=1)

X_reduced = pca.fit_transform(X)

print(X_reduced)
-------------------code----------------------

# Important Notes & Best Practices

### 1. Know When Eigen Decomposition Fails

Eigen decomposition requires:

* Square matrices

SVD does not.

Use SVD for general datasets.

---

### 2. Use Pseudoinverse Instead of Inverse When Necessary

If a matrix is:

* Rectangular
* Singular

use:

-------------------code----------------------
np.linalg.pinv()
-------------------code----------------------

instead of:

-------------------code----------------------
np.linalg.inv()
-------------------code----------------------

---

### 3. Understand Concepts Before Memorizing Formulas

Focus on intuition:

* SVD → matrix decomposition for any matrix
* Pseudoinverse → generalized inverse
* Trace → sum of diagonal entries
* PCA → finding important directions in data
